# Teenconomy

Teenconomy는 고등학생을 위한 체험형 경제교육 플랫폼 MVP입니다.

## 배포 방법

이 저장소는 GitHub Pages에서 바로 실행되도록 구성되어 있습니다.

1. 이 압축 파일을 해제합니다.
2. 안에 있는 모든 파일과 폴더를 GitHub Repository에 업로드합니다.
3. GitHub Repository → Settings → Pages로 이동합니다.
4. Source: Deploy from a branch
5. Branch: main
6. Folder: /docs
7. Save를 누릅니다.

## 폴더 구조

```txt
Teenconomy/
├─ docs/
│  ├─ index.html
│  ├─ style.css
│  └─ app.js
└─ README.md
```

## 특징

- GitHub Pages 즉시 배포 가능
- 별도 build 과정 없음
- React/Vite 사용 안 함
- AI 경제이해력 진단
- 맞춤형 경제학습
- 숏폼 경제학습
- 금융상품 비교
- 모의투자
- 성장 리포트
